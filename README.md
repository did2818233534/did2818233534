## Hi there 👋 I'm Didi

一个专注于 **硬核机器人控制** 和 **电机驱动算法 (FOC)** 的嵌入式工程师。

### 🚀 核心研发方向
- 🔭 **机器人控制架构**：深入开发基于 **ROS2** 的高性能机器人控制框架，主攻双电机同步与运动协同。
- 🌱 **电机驱动算法 (FOC)**：精研无刷电机 FOC 算法，并在 **LKS32MC45x** 和 **STM32G4** 系列 MCU 上实现工业级控制逻辑。
- 📐 **状态估计与算法**：应用 **卡尔曼滤波 (EKF)** 进行角度获取，并实现多种 **无感观测器** 方案（滑模/龙伯格等）。

### 📈 最近的技术实践 (Recent Highlights)
* **高性能驱动**：优化了 LKS32MC45x 的 FOC 控制环路，结合卡尔曼滤波显著提升了低速下的扭矩平稳性。
* **数据监控**：自研基于 C++ 的 PCAN 数据采集工具，实现电机相电流与位置数据的实时波形同步（支持 Teleplot 协议）。
* **建模优化**：通过 SolidWorks 与 Xacro 完成了参数化 URDF 建模，支撑 ROS2 仿真环境下的高精度控制实验。
